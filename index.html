Como usar no OBS: adicione uma fonte do tipo Navegador (Browser), cole o link acima, defina Largura 1920 e Altura 1080 e marque “Atualizar o navegador quando a cena ficar ativa”. O fundo é transparente automaticamente.
Testar avisos: use os botões acima para ver como ficam os avisos de gol e substituição direto na pré-visualização. O botão Modo teste abre o overlay com esses botões na própria tela (útil para ajustar a posição no OBS). O link normal do OBS não mostra os botões de teste.
Importante: o link só funcionará no OBS depois que você hospedar este arquivo num site (ex.: GitHub Pages, Netlify, Vercel). Abrindo direto do seu computador (file://) ele só roda na sua máquina. O placar atualiza sozinho a cada poucos segundos.
